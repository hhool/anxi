# anxi

## Description

This is solution for 2000C machine remote control.

## requirements

- CMake 3.22.0
- C++14
  
## get source code

```bash
$ git clone git@github.com:hhool/anxi.git --recursive
```

## build

```bash
$ cd anxi
$ mkdir build && cd build
$ cmake ..
$ cmake --build .
```

## run

```bash
$ ./bin/anxi
```
